# Client

Frontend built with Vite, TypeScript and React. This folder contains the UI for the LangGraph Debugger.

## Quickstart

Install dependencies and run the dev server:

```bash
cd client
npm install
npm run dev

# or using bun
bun install
bun run dev
```

Build for production:

```bash
npm run build
```

See `package.json` for other scripts (lint/test/typecheck).

## Notes
- Uses Tailwind + shadcn-ui components.
- If you use an editor workspace, add recommended extensions to `.vscode/extensions.json`.
